# RifqyWeather

**Cuaca Real-Time + Ramalan 6 Hari + AI Tip – Otomatis Lokasi!**

> **Cek cuaca di mana saja, kapan saja – dengan UI modern & PWA!**  
> Dibuat oleh [Rifqy Aditya](https://rifqydev.my.id)

---

## Fitur Unggulan

| Fitur | Deskripsi |
|------|----------|
| **Cuaca Saat Ini** | Suhu, kelembaban, angin, deskripsi cuaca |
| **Ramalan 6 Hari** | Mulai besok – suhu rata-rata + ikon cuaca |
| **Cari Kota Manual** | Ketik nama kota → langsung muncul |
| **Lokasi Otomatis** | Pakai GPS HP/PC |
| **AI Tip Pintar** | Saran sesuai cuaca (panas, hujan, dll) |
| **PWA (Installable)** | Bisa dipasang di HP seperti app |
| **Dark/Light Mode** | Otomatis ikut tema HP |
| **100% Gratis & Open Source** | Gunakan, modifikasi, bagikan! |

## Cara Pakai

1. Buka [RifqyWeather](https://rydevs29.github.io/RifqyWeather)
2. Izinkan lokasi → **cuaca otomatis muncul**
3. Atau ketik kota → klik **Cek**
4. Lihat ramalan 6 hari + AI Tip
5. **Install di HP**: Klik titik tiga → "Tambah ke Layar Utama"
